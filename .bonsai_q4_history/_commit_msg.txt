Interleaved Q4 weight reads across SIMD lanes for fully coalesced memory transactions. Changed all 9 Q4 kernels from strided access (lane l reads contiguous block at offset l*cols_per_lane, 32-byte inter-lane stride, 12.5% cache line utilization) to interleaved access (lane l reads word at w*32+lane, 4-byte inter-lane stride, 100% cache line utilization). Added per-word scale/bias accumulation since interleaved words span multiple groups. Changed fused norm kernels from padded TG memory layout to linear layout. Result: decode 132.8 → 213 tok/s average (+60%), parity with MLX reference 90%. Prefill also improved 78 → 97 tok/s.
