Cached residual values in a register array during fused norm Phase 1 (sum-of-squares) and reused them in Phase 2 (normalization), eliminating a second device memory read of the residual buffer. Changed all 3 fused norm kernels (Q proj, K/V proj, gate+up+SiLU). For K=2048, each thread caches only 4 floats (16 bytes) — well within register budget. Saves 8KB of device memory re-reads per TG across 84 fused norm dispatches per token (3 kernels × 28 layers). Result: 223.1-223.3 tok/s (up from ~220 baseline), consistent across 2 runs. p50 improved from ~4530μs to ~4477μs. Parity with MLX reference reached 93.2%.
